Couleurs

En dur

  • Red = Rouge
  • Green = Vert
  • Blue = Bleu

En hexadécimal

  • #FF0000 = Rouge
  • #008000 = Vert
  • #0000FF = Bleu

En RGBA pour Red Green Blue Alpha (opacité)

  • rgba(255, 0, 0, 0.95) = Rouge (opacité 95%)
  • rgba(0, 255, 0, 0.9) = Vert (opacité 90%)
  • rgba(0, 0, 255, 0.85) = Bleu (opacité 85%)
