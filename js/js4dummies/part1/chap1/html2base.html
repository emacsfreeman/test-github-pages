Qui fait quoi ?

Dans la programmation Web classique il y a 2 aspects :

  1. La programmation Front End qui s'occupe du côté client, la devanture d'un site web. Il s'agit de ce que vous voyez dans votre navigateur web.
  2. La programmation Back End qui s'occupe du côté serveur. Il s'agit des bases de données (personnelles). C'est précisément cet aspect qui est problématique car les données sont stockées de façon centralisée mais heureusement il existe une solution grâce à la Blockchain (j'ai fait la traduction des sous-titres en français).

La programmation web Front End c'est quoi ?

Bonne question ! Et bien en fait on peut résumer la chose en ce concentrant sur 3 technologies particulières :

  1. HTML
  2. CSS
  3. JavaScript (ou JS pour les intimes)
